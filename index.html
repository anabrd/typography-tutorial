font-family

Default Value

Sets a specific font for the text.

Syntax:

font-family: [some font];

Hello World!

Internal Font

Browser first attempts to render Arial. If not successful, it moves on to Helvetica. When that's not possible either, it renders sans-serif.

Syntax:

font-family: "Arial, Helvetica, sans-serif";

Hello World!

External Font

Uses external font sheet. Referenced in HTML and CSS.

Syntax:

HTML: <link href>

CSS: font-family: 'Montserrat', Helvetica, sans-serif;

CSS Import: @import url("link");

Hello World!

Font-face Rule

Uses locally downloaded font (e.g. .ttf, .otf formats). Referenced in CSS with the font-face rule.

Syntax:

CSS: @font-face {font-family: "Name", src: url(link)};

Hello World!

font-size

Default Value

Defines the size of the text.

Syntax:

font-size: "size + unit";

Hello World!

Units: em

Defined the size of the text based on the parent element.

Syntax:

font-size: "value em";
Parent container: font-size: 20px;
font-size: 1.2em (24px)
font-size: 1.2em (28.8px)
font-size: 1.2em (34.56px)

Units: rem

Defined the size of the text based on the root element (usually html).

Syntax:

font-size: "value rem";
Root element: font-size: 16px;
font-size: 1.2rem (19.2px)
font-size: 1.5rem (24px)
font-size: 1.8rem (28.8px)

Units: %

Defined the size of the text based on the parent element, only in percentages.

Syntax:

font-size: "value %";
Parent container: font-size: 20px;
font-size: 110%
font-size: 120%
font-size: 130%

Units: Absolute Keywords

Define the size of the text based on the parent element, using absolute keywords like large, small, larger, smaller, etc.

Syntax:

font-size: "absolute keyword";
Parent container: font-size: medium;
font-size: small;
font-size: smaller;
Parent container: font-size: medium;
font-size: large;
font-size: larger;
Parent container: font-size: medium;
font-size: small;
font-size: large;
